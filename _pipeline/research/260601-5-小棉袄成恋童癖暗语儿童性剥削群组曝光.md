# 研究：小棉袄成恋童癖暗语儿童性剥削群组曝光 (260601, #5)

## Facts（事实）

**事件时间线：**

- **2026年5月下旬**：网红在微博、抖音等平台曝光，发现一批用户在社交媒体评论区和私密群组中，使用"小棉袄"作为恋童癖暗语。传统词汇"小棉袄"原意指乖巧贴心的女儿，被恶意扭曲为"性侵女儿"的代称。

- **2026年5月27日**：<font color="blue">抖音官方发布紧急声明，确认发现相关违规内容和账号。平台宣布对所有涉及账号和群组进行无限期封禁，删除违规内容和评论。</font>

- **2026年5月29日**：@英国报姐在微博发布相关报道，阐述事件严重性和家长警惕性不足的问题。同日，微博确认开展打击，相关账号和群组被大规模处理。

**犯罪暗语体系：**

犯罪分子使用的黑话包括：
- "穿小棉袄" = 性侵女儿
- "互换小棉袄" = 互换女儿进行性侵
- "借小棉袄" = 借他人女儿进行性侵
- "买小棉袄" = 购买儿童进行性虐待
- "有偿/无偿" = 带有交易或赠送性质的犯罪行为

**受害者特征：**

根据曝光的聊天记录和截图，受害儿童年龄极低，甚至包含8岁以下的幼童。受害者主要是犯罪分子的亲生女儿或其他可获取的女童。

**犯罪范围：**

此类群组、账号和评论活动涉及多个平台，包括但不限于：
- 抖音评论区
- 微博评论区和私密群组
- 其他社交媒体私密群组

根据曝光内容，至少数十至数百个账号和群组参与此类讨论，具体人数和涉及规模尚未有官方统计公布。

**平台和执法回应：**

- 抖音官方：无限期封禁相关账号和群组，下架违规内容和评论，向主管部门报告并配合调查
- 微博官方：同步开展打击行动，相关账号被大规模禁言或永久封禁，内容被删除
- 相关主管部门：已接收平台举报，是否立案侦查的具体进展未对外公布

## Parties（参与方）

**犯罪嫌疑人：**
- 身份：多为普通网民，包括自称是父亲、叔伯等家庭成员
- 行为：在社交媒体上使用暗语讨论、交流、分享对女童的性虐待内容
- 动机：寻求同好、寻求刺激、贩卖儿童或相关内容获利

**受害者：**
- 主要为犯罪嫌疑人的亲生女儿
- 年龄多为幼童至少年儿童（报道确认包含8岁以下儿童）
- 无法自主举报，高度依赖家长监护和社会保护

**曝光者 @英国报姐：**
- 身份：有影响力的微博及多平台内容创作者，拥有约2000万微博粉丝
- 行为：公开发布报道，揭露事件严重性，呼吁家长警惕
- 声明：强调"平台已封禁相关账号和群组，但家长警惕性仍不足"

**平台方：**
- **抖音**：发布声明，宣布执行无限期封禁、内容下架、向主管部门举报
- **微博**：同步采取措施，进行账号处理和内容清理
- 共同声明要点：
  - 高度重视未成年人保护
  - 严厉打击涉及侵害未成年人的内容和行为
  - 相关视频涉及账号和群已无限期封禁
  - 不良内容、评论将按照平台规则从严处置
  - 向有关主管部门报告，配合调查

**公众舆论：**
- 网友普遍表示强烈谴责和愤怒
- 批评批评网络监管机制的虚伪性（平时高压监管，但对儿童犯罪相对滞后）
- 呼吁加强法律处罚力度、改进家长教育和监护机制
- 对儿童保护组织和执法部门提出期望

**相关主管部门：**
- 接收平台举报
- 具体调查进展、立案情况未对外透露

**研究补充说明：**

本事件反映了互联网时代儿童保护的严峻挑战：
1. 犯罪分子利用暗语躲避算法检测和人工审核
2. 平台内容审核能力有限，往往须依赖用户举报
3. 家长和社会的警惕性和教育仍有重大不足
4. 执法部门对网络犯罪的响应速度和公开透明度需要改进
5. 现有法律框架对网络儿童性剥削的定罪、量刑、证据保全等问题仍有空白
