本站记录中国范围内为主的女性权益相关事件：刑事案件、民事纠纷、舆论事件及政府政策。

内容由自动程序收集，经人工修改和审核后发布。

**事件分级**

| 级别 | 说明 |
|------|------|
| S | 政府/国家层面政策或法律 |
| A | 刑事案件；影响极为恶劣的舆论事件 |
| B | 民事案件；影响较大的舆论事件 |
| C | 非官方组织；影响较小的舆论事件 |
| D | 个人行为 |
| M | 正向进展：推进性别平等的政策、执法行动或倡议（含进行中） |
| N | 中立事件（存疑/已获公正解决/相关性不确定）/等待后续 |

首页日历：有记录的日期显示链接，无记录的日期显示距上次记录的天数。
