# Write Notes

Accumulated style and voice guidance for blog-write. Max ~15 entries.

---

[NOTE] 写稿前必须追踪至今日最新进展。搜索各关键人物/机构的最新动态（判决、庭审、逮捕、声明等），将新发现的事实纳入正文，并用 `<font color="blue">` 标注最新进展。不得仅依赖research文件中的信息，research文件可能已过时。

[NOTE] frontmatter的 `date` 字段应填写**最近一次有实质进展的日期**（如最新判决日、庭审日、逮捕日、声明日），而非今日日期。例如事件最新动态是5月4日开庭，则date填2026-05-04，即使写稿是在5月9日。
