# Research: 河南母亲被嫁案 (260326, #7)

## Facts

**核心事实（时间顺序）：**

- **2008年12月3日**：孙艺菲（时年9岁）之母赵芳在平顶山医院病逝。赵芳名下有位于河北石家庄的多套房产、店面及珠宝。
- **2008年12月3日（同日）**：小姨取得孙艺菲的监护权，孙艺菲亲生父亲孙某签署了放弃抚养权的协议（据其本人后来陈述，签署过程并非自愿，受到威胁与压力）。
- **2008年12月17日**（赵芳去世第14天）：有人以赵芳名义伪造委托书，授权小姨处置其石家庄名下房产；两天内，房产完成出售，房款悉数由小姨收取。赵芳名下珠宝、店面等资产亦相继被转移，最终仅剩一间小商铺。
- **2009年1月**：赵芳户籍在未经合规程序的情况下从河北石家庄迁移至河南鲁山县。
- **2009年12月25日**：鲁山县民政婚姻登记系统中出现赵芳的结婚登记记录，登记对象为其亲弟弟赵功品（时任鲁山县委政法委副书记）。结婚登记声明书中"女方"一栏照片并非赵芳本人，而是其舅妈马某某；二人名下还登记有其他子女信息。
- **2013年**：赵芳户籍正式注销。
- **2018年**：孙艺菲年满19岁，前往公证处申请继承母亲遗产，被告知其母于2009年存在婚姻登记记录，无法证明其为独女，遗产继承受阻。案件就此浮出水面。
- **2026年3月3日凌晨**：鲁山县委、县政府发布官方通报，宣布成立由纪委监委牵头、公安局、民政局等多部门组成的联合调查组，对此事进行全面调查。
- **2026年3月9日**：孙艺菲对媒体透露，案件已由平顶山市联合调查组提级接管，不再由县级层面主导。孙艺菲同时公布了控告书，将矛头指向舅舅、小姨，以及参与文件造假、违规操作的相关公职人员。
- **2026年3月9日起**：孙艺菲接受调查组多次问询，据其本人陈述累计做了至少六次笔录；此后被列为"嫌疑人"，须每日签到、保持手机定位开启。其两次申请调查组相关人员回避，均遭拒绝（首次被拒后上诉，3月10日二次被拒）。
- **2026年3月21日**：孙艺菲收到平顶山市联合调查组通知，约定3月23日（周一）下午赴平顶山市信访局与调查组会面。孙艺菲公开表示将依法配合，同时希望得到公正结果。
- **2026年3月22日**：孙艺菲最后一次更新社交媒体，此后至少3天未有新动态，引发外界关注。
- **2026年3月24日**：孙艺菲亲生父亲孙某首度公开发声，确认前妻赵芳遗嘱系伪造，并指出当年放弃抚养权协议系被迫签署。
- <font color="blue">**2026年3月26日（截至本研究时）**：调查结果尚未公开，孙艺菲动态未更新，案件仍处于市级调查阶段。</font>

**财产损失估算**：涉案遗产包括石家庄多套房产、珠宝首饰、店面等，媒体报道估计价值逾亿元，最终仅剩一间小商铺未被转移。

## Parties

**孙艺菲（举报人/受害人）**
- 赵芳独女，母亲去世时9岁，由小姨抚养长大，2018年成年后发现遗产被侵占，遂实名举报。
- 态度坚决，拒绝任何私了："诉求非常清晰：一是查清母亲当年去世的全部真相，二是追回被侵占的所有遗产，三是追究舅舅、小姨以及所有参与造假、违规操作人员的刑事责任，绝不接受任何形式的私了。"
- 因人身安全顾虑，事先立下公证遗嘱，写明若其在无监控区域"神秘失踪"，应申请法医鉴定，不可认定为意外。
- 3月21日起被列为"嫌疑人"，须每日签到、保持手机定位；3月22日后停止更新社交媒体，原因不明。
- 微博账号（名称暂未确认）用于实时更新进展，是案件信息主要来源之一。

**赵功品（孙艺菲舅舅/涉嫌主犯）**
- 赵芳亲弟，案发时任鲁山县委政法委副书记（公职人员）。
- 其妻马某某的照片被用于冒充赵芳进行婚姻登记。
- 案件曝光后，据孙艺菲透露，赵功品多次联系其希望"私了"，提出经济补偿，恳求删除举报视频，均遭拒绝。
- 无公开声明。

**马某某（孙艺菲舅妈/涉嫌同犯）**
- 其照片出现在赵芳与赵功品婚姻登记声明书的"女方"一栏，系直接冒名顶替者。
- 媒体报道称其曾就此作出某种"回应"，但内容无公开引用。
- 无公开声明。

**小姨（孙艺菲监护人/涉嫌同犯）**
- 取得孙艺菲监护权后，涉嫌以赵芳名义伪造委托书出售石家庄房产，收取房款；并参与转移珠宝等遗产。
- 案件曝光后亦联系孙艺菲要求私了，遭拒。
- 无公开声明。

**孙某（孙艺菲亲生父亲）**
- 2026年3月24日首度公开发声，提供两项关键证词：
  1. 关于遗嘱："遗嘱一看就不是孩子她妈妈写的，字迹我太清楚了，一看就是伪造的。"
  2. 关于放弃抚养权协议：当年签署并非出于自愿，系遭威胁后被迫签署，签署后与女儿联系被切断，多次赴豫探视均被拦截，"每次打电话给女儿都被拦截"。

**赵芳（受害人/已故）**
- 孙艺菲之母，2008年12月3日病逝，遗体土葬，未留正规死亡证明，遗嘱亦被认定为伪造。死后身份遭冒用，被伪造结婚登记，遗产被瓜分。

**鲁山县委县政府（地方当局）**
- 2026年3月3日发布官方通报，成立联合调查组；通报称"高度重视……对此事全面调查并依法处理，结果将适时公开"。
- 孙艺菲申请相关调查人员回避，两次均遭驳回。

**平顶山市联合调查组（上级调查机构）**
- 3月9日接手案件，由市纪委监委牵头，取代县级层面主导调查。
- 3月21日正式通知孙艺菲于3月23日下午赴信访局会面。截至本研究完成，调查结果未公开。

**律师评估**
- 陈婧律师（受访律师）："结合目前的信息，孙女士舅舅舅妈二人的行为在刑法上满足诈骗罪的构成要件，涉嫌刑法第二百六十六条诈骗罪。"此外，如有进一步证据，亦可能涉及行贿（赵功品系公职人员）、伪造印章等罪名。

## 后续进展

### 2026年3月30日：平顶山市联合调查组官方通报

平顶山市联合调查组于3月30日正式发布调查结果，主要内容如下：

**调查认定的违法事实：**

- **2008年12月19日**：赵功品利用时任鲁山县委政法委正科级干部职权，隐瞒姐姐赵芳已死亡的事实，通过时任公安局监察室主任李某，安排工作人员徐某某违规为赵芳和孙艺菲办理了户籍准迁手续（将户口由石家庄迁至鲁山）。
- **2009年1月13日**：赵功品为使女友马某艳冒名续缴养老保险、日后冒领养老金，安排马某艳冒用赵芳身份信息，由徐某某违规为其办理了以赵芳为身份的伪造身份证。
- **2009年1月25日**：赵功品持伪造身份证与马某艳前往民政局，以欺骗手段完成婚姻登记。通报称，二人"后因担心事情败露涉嫌违法，未实施顶替续缴及冒领行为"，即从未实际使用冒名身份骗领养老金。

**关于遗产的认定：**

通报称，调查组<font color="red">"暂未发现赵某品、赵某会侵占赵某遗产的情况"</font>。根据调查组的财务核算：赵芳名下房产售价35万元，扣除房贷10.17万元、银行透支2.55万元、医疗费2.44万元等债务后，剩余约24万元，由赵某会（小姨，监护人）用于孙艺菲十余年的抚养、教育、医疗开销。通报注明，孙艺菲与赵某会之间关于具体费用的争议，留待双方协商或通过诉讼解决。

**关于刑事追诉：**

通报确认伪造身份证、结婚证的行为"涉嫌违法"，但认定"已超过追究时效"，<font color="red">不再追究刑事责任</font>。

**对相关人员的处分：**

| 人员 | 处分 |
|------|------|
| 赵功品（前政法委正科级干部） | 开除党籍、政务撤职 |
| 徐某某（经办公安工作人员） | 党内严重警告、降低退休待遇 |
| 栗某（户籍管理部门负责人） | 诫勉处理 |
| 李某（原公安监察室主任，已故） | 不再追责 |

### 通报后舆论反应

通报发布后，网络舆论出现明显分化：

- 部分网友对通报认定"无侵占遗产"持质疑态度，认为处理结果过轻，尤其对刑事追诉时效已过的认定表示不满。
- 另一部分网友在了解财务核算细节（赵芳遗产存在大量债务、剩余款项用于孙艺菲抚养）后，舆论风向发生逆转，出现"白眼狼"等批评孙艺菲的声音。
- 多家媒体评论认为通报所揭示的伪造身份证、冒名办理结婚登记等行为，反映出基层公职人员对行政权力的严重滥用，但对仅予党纪处分、未追究刑责的结果表示遗憾。

### 孙艺菲状态（通报后）

截至本次研究补充时（2026年5月），未检索到孙艺菲本人对通报的公开回应，亦未见其重启社交媒体更新、启动民事诉讼或提出进一步申诉的报道。其是否已解除"嫌疑人"签到管控状态，亦无公开信息。案件遗产部分的民事争议（孙艺菲与赵某会之间的费用核算）依通报提示留待诉讼途径解决，尚无后续进展报道。
