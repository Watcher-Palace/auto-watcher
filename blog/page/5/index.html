男子杀妻后装行李箱抛尸被判无期

概述

2025年7月，广东中山男子张勇（化名）与妻子李敏（化名）因婚外情纠纷发生冲突，张勇用双手掐住李敏脖子致其死亡，随后将尸体装入行李箱，驱车近200公里抛弃于广东英德偏僻水域。一审以故意杀人罪判处张勇无期徒刑，但判决书将被害人反应列为从轻情节，引发争议。中山市检察院决定不予抗诉，被告张勇已上诉，案件处于二审阶段。

时间线

  • 2025年2月12日：张勇（51岁，广东韶关人）与李敏（44岁，湖南常德人）在广东中山办理结婚登记，均为再婚。
  • 2025年7月15日晚：张勇婚外情被李敏发现，李敏欲举报。双方发生冲突，张勇用双手掐住李敏脖子致其死亡，期间未采取任何抢救措施。
  • 2025年7月15日晚至16日凌晨：张勇将李敏尸体装入行李箱，驱车约200公里抛弃于广东英德偏僻水域。
  • 2025年7月16日早：张勇向姐姐交付银行卡和孩子信息，留下“帮我照顾好孩子”后失踪。当晚警方抓获，张勇当场认罪。

被害人父亲李武（72岁）对判决不满，申请抗诉要求从重判处，但中山市检察院决定不予抗诉。李武称：“这么小的事情为什么要杀人。”他对判决书中关于女儿”处理婚姻不当”的表述表示强烈不满。

4男子下过量合成毒品致15岁女童死亡

概述

2023年10月25日凌晨，深圳龙岗区一家酒吧内，梁某某携带工业级1,4-丁二醇原液调制含毒饮品，经多人传递，将毒液混入饮品强迫15岁女童刘某甲服用。刘某甲于当日上午因γ-羟基丁酸中毒抢救无效死亡。2026年4月，深圳法院以过失致人死亡罪判处四名男子有期徒刑1至1年10个月不等，涉事酒吧所属公司赔偿受害人家属123万元。

时间线

2023年10月25日凌晨：梁某某与王某某在深圳龙岗区某酒吧SV55卡座就座，梁某某随身携带约40毫升工业级”滴滴原液”（1,4-丁二醇），将其混入绿茶调制成”滴滴水”。

同日凌晨：酒吧营销人员郑某某邀请酒吧员工、15岁少女刘某甲前往同一卡座陪同饮酒，刘某甲不知晓饮品含有毒液。王某某要求补充原液，酒吧员工罗某朝空扎壶内喷了约30滴原液添入绿茶。刘某甲在骰子游戏期间被迫服下两大杯含毒液的饮品。

2023年10月25日凌晨4时30分许：刘某甲离开酒吧。

2023年10月25日上午7时许：刘某甲出现严重异常症状，经120抢救无效死亡。经鉴定，死者符合因γ-羟基丁酸中毒死亡。

2026年4月：深圳法院作出一审判决。

判决要点

四名被告均以过失致人死亡罪定罪：

  • 梁某某：有期徒刑1年；赔偿受害人家属7万元，获谅解。
  • 王某某：有期徒刑1年10个月；赔偿受害人家属26万元，获谅解。
  • 郑某某：有期徒刑1年7个月；赔偿受害人家属3万元，获谅解。
  • 罗某：有期徒刑1年10个月。

涉事酒吧所属公司另行赔偿受害人家属123万元，四名被告赔偿总额36万元，合计159万元。

媒体争议

多家媒体的相关报道将案件描述为“过量饮用滴滴水”，遭到网络用户批评，认为该表述将责任归咎于受害者。刘某甲不知晓饮品中含有毒液，由他人在骰子游戏期间迫使其服下含毒饮品，并非主动”饮用”。

背景

1,4-丁二醇是γ-羟基丁酸的前体物质，中国将γ-羟基丁酸列为第一类精神药品加以管制，但1,4-丁二醇本身在案发时未被纳入管制清单。刘某甲因家庭管理失位，15岁时已在涉事酒吧从事陪酒工作。

安徽宿松：未婚同居女子剖宫产后半月被男友起诉退彩礼、分摊分娩费，一审判决返还50%彩礼并失去抚养权

概述

安徽宿松县未婚同居女子李捷（化名）于2025年10月剖宫产生下儿子，产后约半个月，男友陈明（化名）于2025年11月向宿松县人民法院提起诉讼，要求返还彩礼及三金、分摊分娩费、月嫂费及奶粉费。2026年2月，宿松县人民法院作出一审判决：驳回分娩费和月嫂费分摊请求，但判令李捷返还50%彩礼共52,484元，承担一半奶粉费3,164.81元，并因重度抑郁症正在接受治疗而失去子女抚养权。案件于2026年4月中旬经网络披露。2026年4月21日，大象新闻记者向宿松县人民法院核实，工作人员证实「经法院调解，案件当事人双方已达成调解协议」，但调解具体内容未公开。4月22日，李捷发表声明表示「目前正在通过合法途径维权，没有放弃」。

时间线

  • 2025年2月：李捷怀孕。
  • 2025年5月：双方按当地习俗举办婚礼，女方父母为婚礼支出约9万元；未办理结婚登记。
  • 2025年10月：李捷在上海剖宫产生下儿子。

  • 2025年10月底至11月初（产后约半个月）：陈明向宿松县人民法院提起诉讼。

  • 2025年12月26日：法院首次开庭。

男方诉讼请求

陈明起诉要求：返还彩礼66,000元；返还”三金”（含22,590元转账及金手镯一只，价值合计约2万余元）；分摊分娩费；分摊月嫂费；分摊奶粉费。

一审判决要点

法院认定彩礼总额为104,968元（含66,000元彩礼现金、22,590元转账”三金”及16,378元金手镯）。

判令李捷返还50%彩礼共52,484元。判决理由：双方虽未办理结婚登记，但已按习俗举办婚礼、以夫妻名义共同生活近一年并生育子女，彩礼款在婚礼筹备及生育过程中必然有所消耗；男方在女方剖宫产术后身体虚弱、产后不久即提起诉讼，「无视女方生育付出和身心创伤，其行为违背公序良俗，存在明显过错」，因此减半返还。

驳回分娩费、月嫂费分摊请求。理由：产检及分娩费用属母亲妊娠期间个人医疗支出，不属《民法典》规定的法定抚养费范围；月嫂服务属提升生活质量的选择性消费，不具备必要性；基于公平原则，要求女方分担上述费用缺乏明确法律依据。

判令李捷承担一半奶粉费，共计3,164.81元。

孩子由陈明直接抚养。判决理由：李捷因重度抑郁症正在接受治疗，「暂不具备稳定抚养条件」。李捷按月支付子女抚养费。

法院对李捷要求陈明返还嫁妆及个人物品（约2万元）一项，建议另案起诉，未在本案处理。

女方支出与损失

李捷方面在整个过程中的支出包括：女方父母婚礼支出约9万元；分娩及医疗费用（含产检，其中部分由医保报销）；月嫂费用；剖宫产造成的身体不可逆损伤；产后确诊重度抑郁症并持续接受治疗。

概述

国家癌症中心2024年发布的2022年全国癌症数据显示，肺癌超越乳腺癌，成为中国女性发病率第一的癌症：女性肺癌新发病例约40.19万，乳腺癌约35.72万。与2020年数据相比，这一反转极为迅速（2020年：乳腺癌约42万，肺癌约28万）。80%至86%的女性肺癌患者为非吸烟者。

主要风险因素

非吸烟女性的主要风险因素：

  • 二手烟暴露：非吸烟女性中暴露率约70%，约占女性肺癌病例的25%。
  • 厨房油烟：油烟含300余种化学物质，包括苯并芘等致癌物。研究显示，通风不畅环境下持续烹饪6小时产生的油烟，其中多环芳烃浓度相当于点燃96支香烟。
  • 室内及环境污染：煤烟、室内PM2.5、氡气、汽车尾气。
  • 家族遗传史：一级亲属中有1人患肺癌，风险增加85%；2人及以上，风险增至2.72倍。

政策响应

现行肺癌筛查建议见国家卫生健康委2024年8月发布的《肺癌筛查与早诊早治方案（2024年版）》，建议50至74岁高危人群进行低剂量螺旋CT筛查。2026年4月2日，国家卫健委同步发布2026年全国肿瘤防治宣传周活动通知及《中国防癌健康生活方式守则（2026年版）》，后者提出15条防癌生活建议。2026年4月17日，国家癌症中心、中国癌症基金会在北京主办第32届全国肿瘤防治宣传周（4月15日至21日）启动会。本届宣传周主题为“早防早筛早治，同心携手抗癌”。针对烟民及二手烟暴露的管控政策未见明显收紧。

河南鲁山：亡母被伪造身份嫁舅舅，揭起遗产争议

概述

2026年2月底，河南平顶山鲁山县人孙艺菲（时年26岁）在网络实名举报：其母赵芳于2008年12月3日病逝，去世仅14天后即有人伪造赵芳委托书出售其石家庄名下房产；2009年1月，赵芳户口被违规迁至鲁山；同年1月25日，舅舅赵功品（时任鲁山县委政法委正科级干部）持伪造身份证，与女友马某艳冒名顶替赵芳完成婚姻登记。孙艺菲2018年成年办理遗产继承时，才经公证员告知母亲”再婚记录”、无法证明其为独女，就此揭开这起跨度18年的身份伪造与遗产争议案。鲁山县3月3日成立联合调查组，随后提级由平顶山市接管。3月30日官方通报确认身份伪造属实，对涉事公职人员予以党纪政纪处分，但认定遗产”暂无侵占证据”、刑事追诉时效已过，不追究刑事责任。

  • 2026年3月9日 — 案件提级，由平顶山市联合调查组接管，不再由县级层面主导。孙艺菲同日公布控告书，指向舅舅、小姨及相关公职人员。
